# Book Reader

一款 VSCode 状态栏书籍阅读插件，支持在编辑器底部状态栏阅读小说/书籍，俗称"摸鱼阅读器"。

## 功能特性

- **状态栏阅读**：在 VSCode 底部状态栏直接阅读，隐蔽性高
- **自动编码检测**：自动识别文件编码（UTF-8、GBK、GB2312 等），无需手动转换
- **快捷键翻页**：`[` 上一页，`]` 下一页
- **关键词跳转**：配置关键词，自动跳转到书中对应位置
- **阅读速度统计**：实时显示当前阅读速度（行/时）
- **进度显示**：支持显示页码、百分比进度
- **老板键**：`Ctrl+M` 一键伪装工作状态
- **颜色自定义**：支持自定义状态栏文字颜色

## 使用方式

### 1. 配置书籍路径

按 `Ctrl+,` 打开设置，搜索 `Book-Reader`，填写本地书籍文件路径：

- **Mac/Linux**：`/Users/name/Documents/book.txt`
- **Windows**：`C:\Users\name\Desktop\book.txt`

### 2. 快捷键操作

> 输入光标需处在编辑器中才能触发翻页快捷键。

| 快捷键 | 功能 |
|--------|------|
| `[` | 上一页 |
| `]` | 下一页 |
| `Ctrl+M`（Mac: `Cmd+M`）| 老板键：伪装工作状态 |

### 3. 颜色自定义

按 `Ctrl+,` 打开设置，搜索 `Book-Reader`：

- **文字颜色**：在 `textColor` 中填入 hex 颜色值，如 `#ff6b6b`
- **主题强调色**：开启 `useProminentColor`，自动跟随 VSCode 主题

两者同时设置时，`textColor` 优先。都不设置则完全跟随主题。

## 常见问题

**Q: 支持哪些文件格式？**

A: 目前支持 `.txt` 格式。编码自动检测，无需手动转换。

**Q: 老板键怎么恢复书籍内容？**

A: 按 `[` 或 `]` 翻页即可恢复显示书籍内容。
