# Book Reader

一款摸鱼书籍阅读插件

## 配置

`ctrl+,` 弹出设置，搜索 Book-Reader 进行相关设置，使用前需配置书籍路径

## 路径格式

- **Mac** or **Linux** : /opt/test/book.txt
- **Win** : C:\\\Users\\\Administrator\\\Desktop\\\name.txt or C:/Users/Administrator/Desktop/book.txt

## 快捷键

> 输入光标需要处在编辑代码状态才能起作用

`[` 上一页
`]` 下一页
`ctrl+m` 老板键(mac 使用 cmd+m)
